# Hangman Game

## Project Description
The Hangman Game is a fun and interactive word guessing game where players try to guess a hidden word by suggesting letters within a limited number of incorrect guesses. The game ends either when the player successfully guesses the word or the number of incorrect attempts exceeds the limit, resulting in a hangman being drawn.

## How to Play
1. Start the game by choosing a word from the predefined list or let the game select one for you.
2. You will see a series of blanks representing each letter of the word.
3. Guess a letter by typing it in; if it's in the word, it will be revealed in the corresponding blank spaces.
4. If the letter is not in the word, an incorrect guess is counted, and a part of the hangman is drawn.
5. You can continue guessing letters until you either guess the word correctly or exhaust your allowed number of incorrect guesses.
